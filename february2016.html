In The News
AWS Lambda SWF
Amazon have now allowed Lambda functions to be triggered from SWF (Simple Work Flow) so, for example, if you have an image upload on your site, you can create a Lambda function to add a watermark to the image. You can then resize it if you want, and this will be triggered when images are uploaded to S3 Buckets so no extra code to run from the instances!

Offloading more and more work from your instances can only be a good thing. With CloudFront, S3 & Lamdba you require the bare minimum resources for the compute side of things. All this means you should have less need to scale solutions on a regular basis.
